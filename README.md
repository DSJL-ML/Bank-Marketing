# Bank-Marketing
Build machine learning algorithms for bank telemarketing

## Software
This project is written in Python 3.5 and requires the installation of the following Python libraries:

keras		2.0.6

matplotlib	2.0.2

NumPy		1.13.1

pandas		0.20.3

scikit-learn	0.19.1

tensorflow	1.2.1
